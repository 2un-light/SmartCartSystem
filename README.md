## 🚀 주요 기능
- 마트에 있는 상품 앱에서 확인
- 쇼핑할 물품 리스트 담기
- 사용자가 담은 물품과 비슷한 카테고리 상품 추천
- 사용자가 쇼핑할 물품을 토대로 최적의 경로 추천
- 상품 바코드를 스캔하면 해당 상품이 장바구니로 옮겨 감
- 바코드 스캔이 완료된 상품끼리 가격 합산, 리스트나 장바구니에 담긴 물품 추가/삭제 가능
